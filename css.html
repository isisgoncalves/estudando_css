CSS

O CSS é responsável pela estilização

Onde tudo começa a ganhar cores e formas.

Através de um Link no arquivo do HTML, todo o projeto é estilizado pelo CSS.
Na mesma pasta, abre-se um arquivo com o nome style.css, é nele que se aplica a linguagem de estilo.
De volta ao HTML, inserir o seguinte Link:
< link rel= "stylesheet" href= "style.css" > (Link:CSS)

Algumas funcionalidades:
< style > (inicio da configuração)

p { (inicio da configuração em tag P)
(em todas as "tags P" aplica-se a seguinte configuração)
color(cor): grey (nome da cor);
font-size (tamanho da fonte): 21px(quantidade em pixels);
} (término dessa configuração)

h1 { (Inicio de configuração em tag h1)
color: blue;
} (término dessa configuração)

< / style > (término da configuração)

id

O id se sobrepõe as demais tags (p, h1, h2, h3...). Sempre que aplicado, as demais tags na mesma configuração, são inutilizadas.

Espaçamento no CSS

tag margin

margin refere-se a margem entre a tag e a borda da página. Que podem ser as seguintes:
margin-top: em cima.
margin-bottom: em baixo.
margin-left: esquerda.
margin-right: direita.

Os valores também são em pixels (px).
Exemplo:

h1 {
margin-top: 70px;
margin-right: 90px;
margin-bottom: 70px;
margin-left: 90px;

Ou em Shorthand, que é uma forma mais curta:

margin: 70px 90px 70px 90px;
Refere-se a valores na seguinte sequência:
top right bottom left
Segue em sentido horário.

Ou
margin: 70px 90px;
O primeiro valor refere-se a top e bottom, enquanto que no segundo valor refere-se a left e right.

Ou
margin: 70px 90px 70px;
O primeiro valor é o top, o segundo valor são as laterais, e o terceiro valor é o bottom.

E fecha-se a tag com } (colchete).
